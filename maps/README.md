1. Open the terminal and install the required packages by running:
   `pip install -r requirements.txt`
2. Go to https://openweathermap.org/api, and create an account to get an API key.
3. Go to _geo.py_, and change the "INSERT YOUR API KEY HERE" string to your own Openweathermap API key.
4. Run _main.py_ with Python.
5. The expected output is an HTML file created in the project directory. Double-click the file to see the web map.
