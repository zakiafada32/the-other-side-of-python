1. Open the terminal and install the required packages by running:
   `pip install -r requirements.txt`
2. Go to https://www.filestack.com/, and create a Filestack account to get an API key.
3. Go to _reports.py_, and change the "INSERT YOUR API KEY HERE" string to your own Filestack API key.
4. Run _main.py_ with Python.
5. The expected output is a PDF created inside _files_ folder.
